$ curl https://api.arewa.ai/inference/v1/chat/completions \
  -H "Authorization: Bearer $AREWA_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
      "model": "Qwen/Qwen3-Next-80B-A3B-Instruct",
      "messages": [
          {
              "role": "user",
              "content": "What is the secret of life?"
          }
      ]
  }'
import os
from openai import OpenAI

client = OpenAI(
    base_url="https://api.arewa.ai/inference/v1",
    api_key=os.environ.get("AREWA_API_KEY")
)

completion = client.chat.completions.create(
    model="Qwen/Qwen3-Next-80B-A3B-Instruct",
    messages=[
        {
            "role": "user",
            "content": "What is the secret of life?"
        }
    ]
)

print(completion.choices[0].message.content)

Nosotros

Arewa AI es una plataforma mexicana que ofrece LLMs open‑source como servicio (LLMs-as-a-service). Nos enfocamos en el mercado hispano de LATAM, con APIs fáciles de integrar.

Fruto de la frustración por depender de plataformas extranjeras. Sin tener control sobre cómo usaban nuestros datos y los de nuestros clientes.

Nuestra misión es simple: democratizar el acceso a la IA generativa para los desarrolladores de habla hispana. Queremos darte las herramientas más potentes del mundo, con la experiencia y el soporte de alguien que sí te entiende.

Estamos construyendo la infraestructura de IA que Latam se merece.
